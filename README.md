# SLT-Usage Checker

![slt-logo][slt-logo]

### Unofficial usage checker for SLT Internet

___

##### Requirements:

* [Python 3][python]
* [Firefox][firefox]
* [GeckoDriver][gecko-driver]

##### Optional Requirements:

* [Termpix][termpix] (For inline captcha)

##### Dependencies:

* [Selenium][selenium]
* [Halo][halo]
* [Colorama][colorama]
* [pyxdg][pyxdg]
* [Pillow][pillow]

___

### Installation

```sh
$ pip install slt-usage
```

___

### Usage

```sh
$ slt
```
